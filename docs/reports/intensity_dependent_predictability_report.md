# 高震度域における予測困難性と非線形性の検討

## 目的

震度が大きいほど、地盤応答、震度換算、破壊過程、局所的な増幅・減衰、観測点近傍の条件が非線形に効き、現在の「距離減衰 + 地盤増幅補正 + 空間補間」型の方法では予測が難しくなる可能性がある。本レポートでは、保留観測点の誤差を観測震度階級別に集計し、高震度側で予測困難性が増すかを確認した。

## 解釈

この結果は、「高震度ほどこの方法での予測が難しい」という懸念を支持する。ただし、これだけで地盤の非線形応答そのものを直接証明したとは言えない。理由は以下である。

1. 震度5弱以上は階級幅が0.5刻みであり、同じ連続震度誤差でも階級一致率は低くなりやすい。
2. 高震度点は数が少なく、特に震度6強・7では統計的不確実性が大きい。
3. 現在の補間は平滑化を伴うため、局所的な最大震度を構造的に低く見積もりやすい。
4. 地盤非線形性、断層近傍効果、破壊指向性、盆地端部効果、観測点設置条件が混在しており、単独要因に分解できていない。

したがって、最終稿では「震度が大きいほど非線形性が大きい」と断定するより、次のように書くのが妥当である。

> 高震度観測点では、連続震度誤差、階級誤差、過小評価率がいずれも増加し、現在の距離減衰・地盤増幅補正・空間補間に基づく方法では局所的な強震ピークを再現しにくい。この傾向は、強震時の地盤応答の非線形性や震源近傍効果を含む未モデル化要因の寄与と整合的であるが、本解析だけで個別の物理要因を識別することはできない。

## 次に必要なモデル化

高震度域を本格的に扱うには、以下を導入する必要がある。

1. 震度またはPGVの予測分散を震度レベルに依存させる異分散モデル。
2. 強震点での過小評価を明示的に補正する分位点回帰または極値寄りの損失関数。
3. 地盤増幅率を固定値ではなく、強震時の非線形応答を含む確率変数として扱うモデル。
4. 断層面距離、破壊指向性、震源近傍飽和を含むGMPE基準場。
5. 平均的な補間精度ではなく、強震ピーク検出率と過小評価率を主評価指標にする設計。
